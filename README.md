# 🛒 SwiftCart

SwiftCart is a fully functional e-commerce web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Dynamic product rendering using JavaScript & JSON
- Modular codebase using ES6 `import/export`
- OOP-based cart and order management
- Add to cart, quantity update, and item removal
- Real-time price calculation (subtotal, delivery, total)
- Checkout flow with order summary and order history
- Persistent cart and orders using LocalStorage
- Async/Await-based API integration
- Fully responsive UI for all devices

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage
- REST APIs
- JSON

## 📌 Architecture Highlights

- Object-Oriented Programming (OOP)
- Modular and reusable JavaScript components
- Separation of concerns for scalability
